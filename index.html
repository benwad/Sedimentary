What is Sedimentary?

Sedimentary is a Mac app for keeping track of your most commonly-used sed patterns.

sed commands are notoriously confusing to look at - once you've got a pattern right, you want to keep it around in case you want to use it again. Sedimentary lets you store a table of patterns, along with their flags and any helpful notes you make about them. You can then specify an input and output file and run the command from within the GUI.

What is sed?

sed stands for 'stream editor' - it is a powerful tool used to manipulate text coming from an input stream. sed is generally run from a UNIX command line - the user specifies a pattern which acts as a rule for processing the input stream. The stream can then be output to the standard output, or an output file.
